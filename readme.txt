## IP Geolocation API
URL: https://ip-api.com/
API til at tjekke IP addreser. 

## JSONPlaceholder
URL: https://jsonplaceholder.typicode.com/
Et gratis REST API, der tilbyder dummy data for forskellige ressourcer som brugere, posts, kommentarer osv. Du kan bruge det til at hente data, f.eks. brugerlister eller blogindlæg.


## RandomUser.me
URL: https://randomuser.me/
Dette API genererer tilfældige brugerprofiler, som du kan bruge til at få data som navne, adresser, billeder osv. Det er fantastisk til at generere falske brugerdata.


## OpenWeatherMap (Gratis version)
URL: https://openweathermap.org/api
Tilbyder realtidsvejrinformation. Der er en gratis version, der giver adgang til basisoplysninger om vejr for byer rundt om i verden. Dette kan bruges til at hente data og skabe små vejr-relaterede projekter.


## BoredAPI
URL: https://www.boredapi.com/
Et simpelt API, der returnerer forslag til aktiviteter, du kan lave, hvis du keder dig. Dette kan bruges til sjove projekter eller små opgaver.


## CoinGecko API
URL: https://www.coingecko.com/en/api
Dette API giver kryptovaluta data, såsom priser og markedsdata. Det er gratis og meget brugervenligt.


## Cat Facts API
URL: https://catfact.ninja/
Dette API returnerer tilfældige kattefakta, hvilket kan være en sjov måde at inkludere data på i et projekt.
